# Superstore Sales & Profit Analytics Dashboard

## Overview
This project presents an interactive Power BI dashboard built using the Superstore dataset. The goal was to analyze sales performance, profitability, regional trends, monthly sales patterns, and loss-making sub-categories through clear KPIs, slicers, DAX measures, and business insights.

## Tools Used
- Power BI
- DAX
- Data Visualization
- Business Analysis

## Dashboard Pages
### 1. Sales Overview Dashboard
Analyzes total sales, total profit, total quantity, sales by category, sales by region, and monthly sales trends.

## Key Insights
- Technology is the top-performing sales category.
- West region records the highest sales and profitability performance.
- Monthly sales show an upward trend toward 2017 with fluctuations.
- Technology generates the highest total profit.
- Tables show negative profit and may need pricing or cost review.

## Skills Demonstrated
- Built interactive dashboards using Power BI.
- Created KPI cards and slicers for dynamic filtering.
- Used DAX to calculate profit margin.
- Analyzed sales, profit, regional trends, and sub-category performance.
- Translated raw data into business insights and recommendations.
